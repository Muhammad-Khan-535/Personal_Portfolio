HTML is the foundation of every web page. It provides the basic structure and content using elements such as headings, paragraphs, images, and links. HTML tells the browser what to display but not how it should look.

SS is used to style and layout web pages. It controls how HTML elements are displayed like colors, fonts, spacing, positioning, and responsiveness. Without CSS, all web pages would look plain and unstyled.

JavaScript is a scripting language that adds interactivity and dynamic behavior to websites. It allows developers to create features such as interactive forms, animations, real-time updates, and much more. JavaScript runs in the browser and can respond to user actions like clicks or key presses.
React (2024---)
React is a modern JavaScript library developed by Facebook for building fast and interactive user interfaces. It is especially useful for building single-page applications (SPAs), where content updates dynamically without reloading the entire page.
